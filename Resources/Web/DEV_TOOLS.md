# Browser Dev Tools

Get yourself familiarized in Dev Tools! Almost all of the modern browsers offer developer tools to make development and debugging easy for web applications.

Getting yourself familiarized in one browser is enough as they are pretty similar between others.

1. [Chrome Dev Tools Crash Course](https://www.youtube.com/watch?v=x4q86IjJFag) - Video tutorial on Chrome Dev Tools
2. [Chrome Dev Tools](https://developers.google.com/web/tools/chrome-devtools/) - Official Chrome Dev Tools guide
3. [Firefox Dev Tools](https://developer.mozilla.org/en-US/docs/Tools) - Official guide for Firefox Dev Tools
